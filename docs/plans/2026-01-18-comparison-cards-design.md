# 齿轮箱对比卡片设计

> **日期**: 2026-01-18
> **状态**: 已确认，待实现
> **适用范围**: 移动端和桌面端

## 概述

为选型结果添加多结果对比功能，用户可选择2-3个齿轮箱生成并排对比卡片，直观比较核心参数、价格和配套设备。

## 设计目标

1. 支持选择2-3个结果进行对比
2. 并排显示便于横向比较
3. 高亮最优值，标记差异
4. 支持从对比视图直接选定结果

## 用户操作流程

```
选型结果页
    │
    ▼
[加入对比] 按钮（每个结果卡片上）
    │
    ▼
底部浮动栏显示 "已选 2/3"
    │
    ▼
点击 [查看对比]
    │
    ▼
弹出对比卡片视图
    │
    ▼
[选择 XXX] 确认并返回
```

## UI 设计

### 1. 加入对比按钮

在每个结果卡片上添加：

```
┌─────────────────────┐
│ HCM400A             │
│ ...参数信息...       │
│                     │
│ [查看详情] [+ 对比] │  ← 新增对比按钮
└─────────────────────┘
```

**按钮状态：**
- 默认：`[+ 对比]` 灰色边框
- 已选：`[✓ 已选]` 主色填充
- 已满(3个)：其他按钮禁用

### 2. 底部浮动对比栏

选择第1个后出现：

```
┌─────────────────────────────────────┐
│ 📊 已选 2/3    [清空]  [查看对比 →] │
└─────────────────────────────────────┘
```

**样式：**
- 固定在底部
- 背景半透明毛玻璃效果
- 安全区域适配

## 对比参数

| 参数 | 说明 | 高亮规则 |
|------|------|----------|
| 功率范围 | minPower - maxPower kW | - |
| 转速范围 | minSpeed - maxSpeed rpm | - |
| 匹配速比 | 选中的速比 | - |
| 传递能力余量 | margin % | 最高值绿色 |
| 参考价格 | ¥xxx,xxx | 最低价绿色 |
| 联轴器型号 | 匹配的联轴器 | - |
| 备用泵 | 型号或"不需要" | - |

## 交互细节

### 选择操作

| 操作 | 效果 |
|------|------|
| 点击 [+ 对比] | 添加到对比列表，按钮变为 [✓ 已选] |
| 点击 [✓ 已选] | 从对比列表移除 |
| 已选3个时点击其他 | 提示"最多选择3个" |
| 点击 [清空] | 清除所有已选 |

### 对比视图

| 操作 | 效果 |
|------|------|
| 左右滑动 | 切换查看的结果（竖屏3个时） |
| 点击 [选择 XXX] | 选定该齿轮箱，关闭对比，返回结果页 |
| 点击 [✕] 或外部 | 关闭对比视图 |
| 横屏旋转 | 自动切换为3列布局 |

### 高亮规则

- **绿色 ✓**: 该参数为最优值
- **橙色 ⚠**: 该参数明显差于其他（如余量<10%）
- **普通**: 参数相近，无特殊标记
