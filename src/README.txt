# 船用齿轮箱选型系统

这是一个基于React的船用齿轮箱选型系统，用于根据主机参数选择合适的齿轮箱、高弹性联轴器和备用泵，并生成报价单。

## 功能特点

- 导入和管理齿轮箱、高弹性联轴器和备用泵数据
- 根据主机功率、转速和目标减速比选择最合适的齿轮箱
- 自动匹配最适合的高弹性联轴器和备用泵
- 生成完整的技术配置方案
- 提供报价单生成和导出功能

## 支持的齿轮箱系列

- HC系列（中小功率）
- GW系列（大功率）
- HCM系列（轻型高速）
- DT系列（电推）
- 双速系列
- GC系列（变距桨）
- 2GWH系列（双机并车）
- 混合动力系列

## 安装和运行

1. 克隆或下载代码库
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm start`
4. 打开浏览器访问：`http://localhost:3000`

## 数据导入

系统支持通过Excel文件导入产品数据。Excel文件应包含以下工作表：

- HC系列齿轮箱
- GW系列齿轮箱
- HCM系列齿轮箱
- 高弹性联轴器
- 备用泵

每个工作表应包含对应产品的完整参数。

## 使用流程

1. 导入产品数据或使用内置测试数据
2. 输入主机参数和齿轮箱需求
3. 执行选型计算
4. 查看选型结果
5. 生成报价单
6. 导出报价单到Excel

## 依赖项

- React 18.2.0
- React Bootstrap 2.9.0
- Bootstrap 5.3.2
- xlsx 0.18.5
- file-saver 2.0.5

## 开发

### 目录结构

```
src/
  ├── data/                # 数据文件
  │   ├── initialData.js   # 初始基础数据
  │   ├── testData.js      # 测试数据
  │   └── sampleInputTestCase.js # 示例输入数据
  ├── utils/               # 工具函数
  │   ├── dataImport.js    # 数据导入功能
  │   ├── selectionAlgorithm.js # 选型算法
  │   └── quotationGenerator.js # 报价单生成器
  ├── App.js               # 主应用组件
  ├── App.css              # 应用样式
  ├── index.js             # 入口文件
  └── index.css            # 全局样式
```

### 构建生产版本

```
npm run build
```

构建后的文件将位于 `build` 目录中。
